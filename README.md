# agent-skills

A small collection of reusable agent skills.

## Install a skill

Use [`skills.sh`](https://skills.sh) via `npx`:

```bash
npx skills add Svanmark/agent-skills --skill <skill>
```

Example:

```bash
npx skills add Svanmark/agent-skills --skill do-the-needful
```

Another example:

```bash
npx skills add Svanmark/agent-skills --skill skip-that-standup
```

## Available skills

- `do-the-needful`
- `skip-that-standup`
